Comment installer les fichiers de mon archive sur un serveur local XAMPP:

- Premièrement, installer XAMPP.   
- Ensuite copier mon dossier (le site web complet) sae105 dans : C:\\xampp\\htdocs\\   
- Par la suite, il faut lancer le panneau de contrôle XAMPP et démarrer le serveur Apache.   
- Autoriser l’accès aux port 80 et 443   
- Et pour finir, on peut ensuite accéder à notre site avec l’adresse : [http://localhost/sae105](http://localhost/sae105)
